# Recycle Software

Aplicação web client-side para gestão de compra e venda de materiais recicláveis.

## Recursos
- Cadastro de compra e venda com geração automática de ID de cliente (8 caracteres alfanuméricos).
- Formulário completo com campos solicitados.
- Grade automática de até 15 linhas de itens.
- Busca de clientes/transações.
- Impressão em A4 com segunda via para assinatura.
- Aba de resumo com gráficos (barras e pizza).
- Exportação de dados em CSV compatível com Excel.
- Suporte a múltiplos idiomas: PT, ES, EN, JA, VI, UR (Pakistan), ZH, SI (Sri Lanka).

## Como executar
Abra o `index.html` no navegador ou rode um servidor local:

```bash
python3 -m http.server 4173
```

Depois acesse `http://localhost:4173`.
